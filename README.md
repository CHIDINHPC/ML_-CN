## 📌 Giới thiệu

**Smart Electrical Monitoring System** là hệ thống mô phỏng giám sát điện năng theo thời gian thực (Realtime Monitoring) kết hợp với trí tuệ nhân tạo (Machine Learning) nhằm:

* 🔍 Theo dõi thông số điện (Voltage, Current, Temperature)
* ⚠️ Phát hiện sớm nguy cơ sự cố thiết bị
* 📊 Hiển thị dữ liệu trực quan dạng Dashboard (SCADA-style)
* 🤖 Dự đoán sự cố dựa trên mô hình học máy

👉 Ứng dụng trong:

* Tòa nhà thông minh (Smart Building)
* Hệ thống điện công nghiệp
* Trung tâm điều khiển (SCADA Simulation)

## 🧠 Công nghệ sử dụng

| Công nghệ        | Mô tả                  |
| ---------------- | ---------------------- |
| 🐍 Python        | Ngôn ngữ chính         |
| 🎨 Streamlit     | Xây dựng giao diện web |
| 📊 Pandas        | Xử lý dữ liệu          |
| 🤖 Scikit-learn  | Machine Learning       |
| 📈 Plotly        | Biểu đồ & Gauge        |
| 💾 Excel Dataset | Dữ liệu đầu vào        |

## 📊 Dataset mô tả

Dữ liệu bao gồm các thông số điện thực tế:

| Thuộc tính       | Ý nghĩa         |
| ---------------- | --------------- |
| Voltage (V)      | Điện áp         |
| Current (A)      | Dòng điện       |
| Temperature (°C) | Nhiệt độ        |
| Load (%)         | Tải             |
| Power Factor     | Hệ số công suất |
| Frequency (Hz)   | Tần số          |
| Fault            | Trạng thái      |
| Fault_Code       | Loại sự cố      |
